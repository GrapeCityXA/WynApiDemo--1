如果想要在业务系统中集成某个仪表板的设计页面，可采用URL集成方式。要点是：修改仪表板查看页面的集成URL，将 /view/ 改为 /edit/ 

例如：查看页面的集成URL字串如下：

http://localhost:51980/dashboards/view/5b809a606d502b0031deb3cd/?mode=server&display=singlepage&theme=default&token=2cb6c4d1717902b9a1b46f9c58cfae6de02d5a9982e458d190a474da36748630

将 /view/ 改为 /edit/ 之后，URL成为：

http://localhost:51980/dashboards/edit/5b809a606d502b0031deb3cd/?mode=server&display=singlepage&theme=default&token=2cb6c4d1717902b9a1b46f9c58cfae6de02d5a9982e458d190a474da36748630

将这个URL字串设置为业务系统页面中某个iframe的src属性或者超链接的href属性，即可将设计页面集成到业务系统中。
